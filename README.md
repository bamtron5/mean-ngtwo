# NG2 Mean Stack on TypeScript

## Release
Pre-alpha | Dev only. Contribution welcome.

## Run directions
`npm install`

then:
`npm run dev`

##Package Execution Scripts
- `npm run serve`
- `npm run dev`
- `npm run watch`

## Includes	
* Authentication w/ JsonWebTokens
* Components:
	- Nav Component
	- Login Component
	- Todo List Component
	- Profile Component 
* Mongo Seeds (todo model, users model)
* My NG Service Boiler Plate
* Express API routes
* Static Routing
* Jasmine Testing

## Future Support

	- `heroku deploy`
